say server
==========
use say and saykana via http


Dependencies
------------

* Mac OSX
* Sinatra
* MeCab
* [saykana](http://www.a-quest.com/quickware/saykana/)


Run Server
----------

config

    % cp sample.config.yaml config.yaml

then, edit it.


Install Dependencies

    % gem install bundler
    % bundle install


run

    % ruby development.rb

for passenenger, use config.ru
